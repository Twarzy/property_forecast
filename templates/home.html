Property price prediction

by Bartosz Kowalik

Predict using only Google Sheets

Click link below to access our sheet with implemented script, that connects it with our prediction server.
You can make property price predictions directly in Google sheets.

Google Sheets

How to start predictions using Google Sheets:
  • ▸Open Google Sheets, using green button above.
  • ▸Sign in, with you google account.
  • ▸"Custom Menu" appears in the navbar, next to "Help"
  • ▸Click "Custom Menu" ▹ "makePrediction"
  • ▸Now You have to authorize document script: (only the first time)
  •  ▪ Confirm your google account.
  •  ▪ Click "Advanced" ▹ Go to Property Price Prediction by Bartosz Kowalik
  •   Don't worry about "unsafe" notation. It's just because Goggle can't verify custom scripts
  •   You can always access this script clicking "Extensions" ▹ "Apps Scripts" to make sure it is safe
  •  ▪ Finally press "Allow" and it's done.
  • ▸Price prediction will appear in "predict" column (You don't have to clear it before prediction).
You can modify data, add or delete rows in sheet. Format of input data is explained below.
Use "makePrediction" how many times you want.

Data used to train machine learning model was in polish and input data is expected to be in that language as well.
  • ▸area:
  •  ▪ Property area
  •  ▪ Float number
  • ▸rooms:
  •  ▪ Number of rooms
  •  ▪ Integer number
  • ▸location:
  •  ▪ Location of property
  •  ▪ Location from general to specific, coma separated. Correct capital letters expected.
  •  ▪ Ex. Province,County,City,District
  •  ▪ Only part of location can be used ex. Province,City or City
  • ▸floor:
  •  ▪ Property floor number
  •  ▪ Integer number
  • ▸floors_in_building:
  •  ▪ Number of all floors in building
  •  ▪ Integer number
  • ▸rok budowy:
  •  ▪ Build year
  •  ▪ Integer number
  • ▸is_primary_market:
  •  ▪ Is primary market
  •  ▪ Boolean: True or False
  • ▸rodzaj zabudowy:
  •  ▪ Type of building
  •  ▪ Choose from: "blok", "apartamentowiec", "kamienica", "dom wolnostojący", "szeregowiec", "plomba", "loft"
  • ▸okna:
  •  ▪ Type of windows
  •  ▪ Choose from: "plastikowe", "drewniane", "aluminiowe"
  • ▸stan wykończenia:
  •  ▪ Finish condition
  •  ▪ Choose from: "do zamieszkania", "do wykończenia", "do remontu"
  • ▸ogrzewanie:
  •  ▪ Type of heating
  •  ▪ Choose from: "miejskie", "gazowe", "inne", "kotłownia", "elektryczne", "piece kaflowe"
  • ▸forma własności:
  •  ▪ Form of the property
  •  ▪ Choose from: "pełna własność", "spółdzielcze własnościowe", "spółdzielcze wł. z kw", "udział"
  • ▸system alarmowy:
  •  ▪ Alarm system
  •  ▪ Boolean: True or False
  • ▸rolety antywłamaniowe:
  •  ▪ Anti-burglary blinds
  •  ▪ Boolean: True or False
  • ▸drzwi / okna antywłamaniowe:
  •  ▪ Anti-theft doors/windows
  •  ▪ Boolean: True or False
